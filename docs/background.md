# SerialLeg 机器人背景规格

机器人物理规格、电机参数和速度能力，供训练指令范围、奖励设计和 sim2sim 验证参考。

## 机体结构

- 自由度：6 DOF（左腿 lf0/lf1/l_wheel + 右腿 rf0/rf1/r_wheel）
- 轮径：**R = 59 mm**（直径 118 mm，来自 MJCF `wheelRadius: 0.059 m`）
- 腿部传动：四连杆机构，膝关节通过驱动杆 AB → 连杆 BC → 小腿 CD 传动
- 轮子安装：小腿末端，接地点固定作为参考坐标系原点

## 轮子最大线速度

基于 M3508-Hexroll 输出轴转速 × 轮径：

| 工况 | 输出轴转速 | 线速度 |
|------|-----------|--------|
| 空载理论上限 | 61.5 rad/s | **3.63 m/s** |
| 额定扭矩（连续推进） | 20.5 rad/s | **1.21 m/s** |
| RL 训练 action 上限（scale=20） | 20.0 rad/s | **1.18 m/s** |

额定扭矩下的输出轴转速由线性 T-N 关系推导：

```
ω_rated = ω_no_load × (1 − rated_torque / stall_torque)
        = 61.5 × (1 − 2.46 / 3.69)
        ≈ 20.5 rad/s
```

平地连续行驶的速度上限约 **1.2 m/s**。短时冲刺理论可到 **3.6 m/s**，但此时轮子输出扭矩趋近于零。RL 训练的 action scale = 20 rad/s 把轮速目标限制在 **1.18 m/s**，和额定工况基本吻合。训练速度课程最高指令 ±2.5 m/s 超出额定范围，属于短时冲刺区间。
